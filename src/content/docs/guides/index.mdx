# Practice Guides

Practice Guides translate scientific evidence into practical workflows that support health creation in everyday practice.

Unlike Evidence pages, which ask **"What does the evidence show?"**, Practice Guides ask **"How can this evidence be applied?"**

Each guide integrates scientific evidence, clinical expertise, contextual factors and people's preferences to support evidence-informed action.

The Practice Knowledge System recognises that health is ultimately created in people's daily lives. Accordingly, Practice Guides are designed not only for healthcare professionals but also for individuals, families, communities and policy makers.

Each guide typically includes:

- Purpose
- When to use
- Evidence summary
- Practical workflow
- Decision points
- Related interventions
- Related patient resources
- Monitoring and evaluation

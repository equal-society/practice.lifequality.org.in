# Learning Resources

Learning Resources provide educational materials that help individuals, healthcare professionals, researchers and communities understand and apply evidence.

Knowledge becomes valuable only when it can be understood, shared and used. Accordingly, this section complements the scientific content of PKS with practical educational resources suitable for different audiences.

Resources may include:

- Slide presentations
- Infographics
- Videos
- Workshops
- Courses
- Patient education materials
- Professional development modules
